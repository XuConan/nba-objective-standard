# NBA奖项客观数据标准评选技能 - 验证检查清单

## 数据获取层验证
- [ ] 检查 nba_api 数据获取功能是否正常
- [ ] 检查 balldontlie API 数据获取功能是否正常
- [ ] 检查 Basketball-Reference 数据爬取功能是否正常
- [ ] 检查多数据源返回数据格式一致

## 交叉验证层验证
- [ ] 检查多源数据交叉验证逻辑是否正确
- [ ] 检查置信度标记是否正常生成
- [ ] 检查数据差异>10%时是否标注警告
- [ ] 检查缺失指标是否正确降级处理

## 常规赛奖项验证
- [ ] 检查 MVP 评选算法（战绩分60%+高阶数据分40%）
- [ ] 检查 DPOY 评选算法（DBPM、DWS、抢断+盖帽）
- [ ] 检查数据王评选（得分、篮板、助攻、抢断、盖帽）
- [ ] 检查最佳阵容评选（位置分配：2G+2F+1C）
- [ ] 检查最佳防守阵容评选
- [ ] 检查最佳新秀评选
- [ ] 检查最快进步球员评选
- [ ] 检查最佳第六人评选

## 季后赛奖项验证
- [ ] 检查总冠军识别
- [ ] 检查 FMVP 评选（总决赛数据+优先胜方）
- [ ] 检查季后赛 MVP 评选（整个季后赛表现）

## CQI 验证
- [ ] 检查自身统治力计算（常规赛战绩分+季后赛净效率分）
- [ ] 检查对手强度计算（对手平均胜场+SRS）


- [ ] 检查伤病调整分计算
- [ ] 检查传奇性分计算
- [ ] 检查 CQI 结果在 0-100 范围内
- [ ] 检查含金量评级正确

## 导出功能验证
- [ ] 检查 Markdown 文件生成正常
- [ ] 检查文件包含数据源声明
- [ ] 检查奖项对比表格格式正确

## 集成验证
- [ ] 完整流程测试（获取数据→交叉验证→评选→输出）
- [ ] 测试多个赛季数据（如2023、2024）
- [ ] 检查错误处理机制
